Project: Augmented Reality Application

Tools Used: Unity Editor, Vuforia Engine SDK, Web Camera, Smartphone Camera

Design Process - Main Scene

1. Marker chosen: Gearbox Cohort 7 Logo.

2. Image target uploaded to Vuforia Engine Developer Portal.

3. Unity editor first scene named main scene; a plane object designed with the image target as the background.

4.For the user interface, a canvas was added to the hierarchy tab; within the tab add a button (text mesh pro) called “Next Scene”.An image depicting next was downloaded:

Scene 2

1. Plane object made a child of the image target; an art piece added to the albedo component.

2.Once this scene is switched to, the art piece till appear on-top of the image target.

3. Second canvas added with an additional button scripted to exit the app.

Scripting the Buttons

• A scene changer object was designed.

• A scene changer script was created and added to the scene changer object. From here, once a button is pressed, a scene changer function will be called upon according to its functionality.

• Functions defined for scene changing and app exit.
